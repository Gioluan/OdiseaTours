# Odisea Tours — Brand Guidelines

Official brand assets for Odisea Tours (Ground Agents Solutions S.L.).

## Logos

Located in `logos/Logo Files/`:

- **SVG** — vector, use for web and print
- **PNG** — raster, for quick use
- **PDF** — print-ready
- **Favicons** — Android, Browser, iPhone

Variants: Color (default), Black, White, Color-with-background.

## Colors

| Role       | HEX       | RGB                | CMYK                  |
| ---------- | --------- | ------------------ | --------------------- |
| Light / BG | `#f6f6f6` | 246, 246, 246      | 0, 0, 0, 3.5          |
| Accent     | `#ffb400` | 255, 180, 0        | 0, 29, 100, 0         |
| Dark / Ink | `#111111` | 17, 17, 17         | 0, 0, 0, 93           |

## Typography

- **Wordmark** — Biko Bold
- **Slogan** — Avenir Heavy

## Usage

- Keep clear space around the logo equal to the height of the "O".
- Do not recolor, stretch, or add effects to the logo.
- Prefer SVG on web; use PNG where SVG is not supported.
- On dark backgrounds use the white variant; on light backgrounds use the color or black variant.
